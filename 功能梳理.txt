功能梳理

1.首页

定位用户位置，手动选择城市（目前仅限绵阳市，所以页面上暂时隐藏）

广告banner（后台增加系统设置相关内容，可以设置这里）
{
	后台功能: 
	1.添加（填写标题，上传图片，设置跳转连接，设置状态,设置显示位置：如每个分类的banner不一样，首页banner又不一样） 
	2.列表展示，分页   
	3.编辑
}


搜索{
	1.记录关键词
	2.搜索结果（选择最新的30条数据， 默认时间排序， 有条件按条件排序， 每次加载5条，最多加载6次则显示没有更多）
}



列表{
	1.默认显示最新的
	2.有查询条件（如分类）或排序条件（时间，金额），按条件查询排序显示（每次10条，最多可以加载6次）
}


2.详细页面
后台： {
	1.文章添加时，加位置信息（推广店铺的位置信息）
	2.文章添加时，添加banner图功能
	3.添加有效时间范围设置
    4.增加收藏字段 
}

前台：{
	1.各部分内容展示，banner滑动效果
	2.有效时间等的验证及后面对分享，点击等的处理
	3. 数据统计（分享，收藏，点击等）
}



3.个人中心
收藏统计
分享统计
收入明细{
	1.按天统计点击数，统计金额
	2.默认显示最近一周的收益情况
	3.可以点击显示本月收益，上月收益
}

新手攻略，问题帮助，用户须知{
	目前就做简单版本，三个单独的页面
}

money处理{
	usersmoney表处理

	1.点击分享时
	判断updatetime（{同一天：moneytoday+，moneythismmonth+，moneytotal+}{不是同一天：计算所有的money数据}）

	2.每天0点1分，计算所有的数据，timer.php
}


money审核{
	1.timer在每月开始的1号凌晨00：30分开始

	2.通过clickog和clickcount的数据作统计， 结算结果写入数据表moneycount，
	（moneycount的结果正确与否通过clicklog和count计算验证）
	（leftmoney 可以通过moneycount与moneypay计算 ）

	3.所有的money，可以通过moneylog统计验证
}
